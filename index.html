Coursework

Labs

These lab activities helped me build my foundation in mobile application development, from interface design to data storage using Room.

Lab 1

Lab 1: UI

This lab focused on creating a basic user interface with proper structure, layout arrangement, and readable visual design.

Reflection: This lab helped me understand that a good application starts with a clear and simple interface. I learned how spacing, alignment, and component placement can affect the user experience. It also reminded me that design should be both attractive and functional.

Lab 2

Lab 2: Interaction

This lab introduced interactive elements that allow users to perform actions and receive feedback from the application.

Reflection: Through this lab, I learned how interaction makes an application feel more useful and responsive. I became more confident in handling buttons, user input, and simple event actions. This experience showed me the importance of giving users clear feedback after every action.

Lab 3

Lab 3: Material Design

This lab explored Material Design principles to create a more consistent, modern, and user-friendly mobile interface.

Reflection: This lab helped me understand how design guidelines can make an app look professional and easier to use. I learned to apply consistent colours, components, and visual hierarchy. It also taught me that small design improvements can make a big difference in user confidence.

Lab 4

Lab 4: Navigation

This lab focused on moving between screens and organising application content through proper navigation flow.

Reflection: I learned that navigation is important because users need to move through the application smoothly without confusion. This lab helped me understand how screens connect and how information should be arranged logically. It also improved my ability to plan app flow before coding.

Lab 5

Lab 5: Room

This lab introduced local database storage using Room, allowing application data to be saved and managed more effectively.

Reflection: This lab taught me how important data persistence is in real mobile applications. I learned how Room helps store, retrieve, update, and delete data in a structured way. Although database concepts were challenging at first, the lab helped me connect theory with practical app development.

Application Development

Projects

The projects show the development progress of VIORA CYCLE, an application idea aligned with SDG 3 by supporting personal health awareness and well-being.

Project 1

VIORA CYCLE: Lab 4 + Working Features

Problem Statement: Many users need a simple and organised way to track cycle-related information and understand their personal health patterns. VIORA CYCLE aims to provide a clear mobile experience that supports awareness, planning, and healthier self-management.

Lessons Learned: Project 1 helped me combine navigation, interface design, and working features into one complete application flow. I learned the importance of testing each screen carefully so users can move through the app without errors. This project also improved my confidence in turning a concept into a functional prototype.
